This is a transliteration tool which facilitates transliteration from Unicode Devanagari to various transliteration schemes such as: WX-alphabetic being used ay the Akshar Bharati group, SLP1 being used by the Sanskrit Library Project at Brown, Velthuis used in European countries, Harvard-Kyoto, Itrans and Roman transliteration, and vice versa.
The interface and the shell programs are written by Karnunakar.
Most of the transliteration programs are written by Amba Kulkarni.
All the programs are available under GPL. Contact ambapradeep@gmail.com for the source code.
