What does each row specify?

  • Row 1: Sandhi Split (pada paatha) Sanskrit text
  • Row 2: Original Sanskrit Text (samhitaa paatha)
  • Row 3: Morphological Analysis in the given context
  • Row 4: Hindi meanings of the word
  • Row 5: English Meanings of the word

    Top

    How to control number of rows?

    Use the Show/Hide button in the bottom frame to control the display of rows.

    Top

    What do the colors specify?

    The colors in the output indicate the vibhakti of the word(s). Following color scheme has been used.

    prathama vibhakti
    dvitiyaaa vibhakti
    tritiyaa vibhakti
    caturthi vibhakti
    panchami vibhakti
    shashthi vibhkati
    saptami vibhakti
    sambodhana
    verb

    Top

    What do the numbers in the output stand for?

    The numbers in lightgreen strip indicate the Paragraph, Sentence, and Row numbers. For example 10.5.G stands for the Gth row of the 5th sentence in 10th paragraph. One can use paragraph and sentence number information to jump to a particular location using GOTO button in the top frame. Thus if you want to jump to say 5th sentence in the 10th paragraph, just type in 10.5 and click on GOTO button.
