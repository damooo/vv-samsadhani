सन्धि-विच्छेदिका (Sandhi-Splitter) ☞ सन्धि-विच्छेदिका इत्युपकरणम् उपयोक्तुं ध्यातव्याः अंशाः जुगुप्सन्ताम्    

Sandhi-Splitter segments a given Sanskrit string into different meaningful components. If the input string is a compound, you may help machine reducing the search space by indicating the type of split as पदच्छेदः. In case the input string is a compound, choose the समासच्छेदः option. In case you are unsure, choose उभयोरपि.

If more than one splittings is possible machine ranks them according to the probabilities. The detailed description of how the splits are ranked is available here. Only the most probable is displayed as a result, and other splits are available on demand. Machine also shows the morphological analysis of each of the components, when a mouse is placed over the word.
