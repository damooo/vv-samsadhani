पद-विश्लेषिका(Morphological Analyser) analyses a Sanskrit word giving its nominal stem (प्रातिपदिकम्) / verbal root (धातुः) along with its various linguistic features such as Lexical-category (जाति), Gender (लिङ्गम्), Number (वचनम्), Case (विभक्तिः), लकारः, Person (पुरुषः), etc. For instance, for the word रामः, it gives the following analysis

राम पुं 1 एक रा1 कर्तरि लट् उ 3 परस्मैपदी अदादिः


    पद-विश्लेषिका (Morphological Analyser)
Encoding For instances: रामः, रमा, गच्छति etc.
