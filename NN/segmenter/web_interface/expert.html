न्यायचित्रदीपिका

A tool to analyse Navya-Nyāya Expressions

संसाधनी-Saṃsādhanī

Department of Sanskrit Studies, University of Hyderabad.

Enter a Navya-Nyāya Expression.
For instance:
in Devanagari: गन्धनिष्ठाधेयतानिरूपिताधिकरणता, or
in WX notation: ganXaniRTAXeyawAnirUpiwAXikaraNawA
A Segmenter splits a given NNE into components using the Nyāya lexicon.
We use SCL segmenter which is a part of Saṃsādhanī a platform for Sanskrit Computational Linguistics
