What isआजीविका ?

आजीविका is the relation which are showing livelihood. Eg. ::

अर्थः :: वणिक् | वर्गः :: वैश्यवर्गः | आपणिक, क्रयविक्रयक, नैगम, पण्याजीव, सार्थवाह, वणिज्, वाणिज, वैदेहक

आजीविका

अर्थः :: वाणिज्यम् | वर्गः :: वैश्यवर्गः | वणिग्भाव, सत्यानृत

In this example it is showing that the relation between वणिक् and वाणिज्यम्, and here it is livelihood relation.
