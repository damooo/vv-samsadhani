What is अवयवः or Meronymy ?

Meronymy relation means, a part of whole. Eg. ::

अर्थः :: वृक्षः | वर्गः :: वनौषधिवर्गः | अगम, अनोकह, कुट, महीरुह, पलाशिन्, पादप, शाखिन्, शाल, विटपिन्, वृक्ष, तरु, द्रुद्रुम, अग, नग, शिखरिन्, अद्रि, विष्टर, धव

अवयवाः (is have parts as)

अर्थः :: तरुमूलम् | वर्गः :: वनौषधिवर्गः | जटा, शिफा, विटप, गुल्म, नेत्र
अर्थः :: शाखा | वर्गः :: वनौषधिवर्गः | लता, शाखा
अर्थः :: वृक्षत्वक् | वर्गः :: वनौषधिवर्गः | वल्क, वल्कल, त्वच्, शल्क
अर्थः :: काष्ठम् | वर्गः :: वनौषधिवर्गः | काष्ठ, दारु
अर्थः :: पत्रम् | वर्गः :: वनौषधिवर्गः | छद, छदन, पलाश, पर्ण, पत्र, दल
अर्थः :: वृक्षफलम् | वर्गः :: वनौषधिवर्गः | फल, सस्य
अर्थः :: पुष्पादिबन्धनम् | वर्गः :: वनौषधिवर्गः | प्रसवबन्धन, वृन्त

Here in this example वृक्षः is having part as तरुमूलम्, शाखा, वृक्षत्वक् etc.
