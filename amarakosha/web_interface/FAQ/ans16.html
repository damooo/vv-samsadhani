What is सम्बन्धितः ?

सम्बन्धितः is relating the things which are relating with any thing. Eg. ::

अर्थः :: अमृतम् | वर्गः :: स्वर्गवर्गः | अमृत, सुधा, पीयूष

संबन्धितः

अर्थः :: देवः | वर्गः :: विशेष्यनिघ्नवर्गः | अमर, निर्जरस, सुमनस्, सुपर्वन्, सुर, विबुध, त्रिदश, त्रिदिवेश, देव, दिवौकस्, अमर्त्य, अमृतान्धस्, अस्वप्न, आदितेय, अदितिनन्दन, आदित्य, लेख, ऋभव, दिविषद्, बर्हिर्मुख, गीर्वाण, क्रतुभुज्, वृन्दारक, दानवारि, देवता, दैवत, दिव्योपपादुक, विवस्वत्, अनिमिष

Here in this example अमृतम् is the thing related with देवः.
