What is अपराजातिः or Hyponymy ?

Hyponymy relation means, verities of a particular thing. Eg. ::

अर्थः :: नदी | वर्गः :: वारिवर्गः | नदी, सरित्, आपगा, ह्रादिनी, निम्नगा, शैवलिनी, स्रवन्ती, स्रोतस्विनी, तरङ्गिणी, तटिनी, धुनी, द्वीपवती, कूलङ्कषा, निर्झरिणी, रोधोवक्रा, सरस्वती, भोगवती, सिन्धु, वाहिनी

अपराजातिः (is have types of)

अर्थः :: गङ्गा | वर्गः :: वारिवर्गः | सुरनिम्नगा, गङ्गा, जह्नुतनया, विष्णुपदी, भागीरथी, भीष्मसू, त्रिपथगा, त्रिस्रोतस्
अर्थः :: यमुना | वर्गः :: वारिवर्गः | कालिन्दी, शमनस्वसृ, सूर्यतनया, यमुना
अर्थः :: नर्मदा | वर्गः :: वारिवर्गः | मेखलकन्यका, नर्मदा, रेवा, सोमोद्भवा
Here in this example गङ्गा, यमुना and नर्मदा are different names of नदी.
