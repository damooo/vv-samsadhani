What is जन्यजनकभावः ?

जन्यजनकभावः is a relation between child and parent. Eg. ::

अर्थः :: जयन्तः | वर्गः :: स्वर्गवर्गः | पाकशासनि, जयन्त

जनकः

अर्थः :: इन्द्रः | वर्गः :: स्वर्गवर्गः | इन्द्र, बिडौजस्, मघवन्, मरुत्वत्, पाकशासन, सुनासीर, वृद्धश्रवस्, पुरन्दर, पुरुहूत, जिष्णु, लेखर्षभ, शक्र, शतमन्यु, दिवस्पति, वृषन्, वृत्रहन्, गोत्रभिद्, सुत्रामन्, वासव, वज्रिन्, बलाराति, शचीपति, सुरपति, वास्तोष्पति, हरिहय, जम्भभेदिन्, नमुचिसूदन, स्वाराज्, मेघवाहन, सङ्क्रन्दन, तुराषा, दुश्च्यवन, आखण्डल, ऋभुक्षिन्, सहस्राक्ष, कौशिक, घनाघन, पर्जन्य, हरि

Here in this relation it is showing thatजयन्त's father is इन्द्र.
